What is StiGame ?

StiGame is a C++ library that wrap SDL2 Functions. This give the opportunity to use SDL2 with the object oriented paradigm. Which you can't do in C. StiGame also include other features like event handling with Observer design patterns, a finite state machine to handle the game state for your application. StiGame also include a Graphical user Interface Stack to create interface rapidly for your application.
